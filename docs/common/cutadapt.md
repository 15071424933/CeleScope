

## Arguments
`--adapter_fasta` addtional adapter fasta file

`--minimum_length` minimum_length

`--nextseq_trim` nextseq_trim

`--overlap` minimum overlap length

`--insert` read2 insert length

`--fq` fq file

`--gzip` output gzipped fastq

`--outdir` output dir

`--assay` assay

`--sample` sample name

`--thread` None

`--debug` debug
