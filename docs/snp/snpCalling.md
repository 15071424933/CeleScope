

## Arguments
`--outdir` Output diretory.

`--assay` Assay name.

`--sample` Sample name.

`--thread` Thread to use.

`--debug` If this argument is used, celescope may output addtional file for debugging.

`--bam` featureCounts bam

`--match_dir` match scRNA-Seq dir

`--genomeDir` genomeDir

`--gene_list` gene_list
